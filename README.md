# To-Do List App

This is a simple To-Do List application built with React. It allows you to add, delete, and manage your tasks.

## Features

- Add new tasks
- Delete existing tasks
- Simple and intuitive user interface

- [npm](https://www.npmjs.com/get-npm) (comes with Node.js)
